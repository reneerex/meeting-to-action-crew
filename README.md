# Meeting-to-Action Crew

A multi-agent AI workflow built using CrewAI and Ollama that converts meeting transcripts into structured project reports.

## Features

- Meeting summarization
- Action item extraction
- Responsibility assignment
- Deadline extraction
- Report generation
- YouTube transcript support

## Tech Stack

- CrewAI
- Ollama
- Qwen3:4B
- Python

## Workflow

Meeting Transcript
↓
Meeting Summarizer
↓
Action Item Extractor
↓
Responsibility Tracker
↓
Deadline Extractor
↓
Report Generator
